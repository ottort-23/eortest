Sale Discount Limit

The Sale Discount Limit module prepared by Aktiv Software will help you put a restriction on the sale discount which can only be posted by authoritative person such as Sales Manager and Sales/User. By default, the module has allowed a limit of providing a maximum of 15% discount for Sales Manager. Sales user can, on the other hand, can provide a maximum of 10% discount.

Configurations

  • Go to Sales > Configuration > Sales Discount Limit

Select Sales User Group and Put Discount in Percentage

Having created a user and configured with access rights of “Sales Manager” and “Discount on Lines” they will get the rights to apply a maximum of 15% discount.

If the user having rights of “Sales Manager” tries to apply discount of more than 15% a warning will pop up as shown in the below image. Similarly, the Sales user if tries to apply discount more than 10% warning will be raised saying “You are not eligible to give discount more than 10%”.
