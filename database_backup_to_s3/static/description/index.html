Auto Backup Database to Amazon S3 bucket

Backup your database directly to Amazon S3 bucket

  • Go to Settings -> S3 Backup -> Configure S3
  • Prior using this module make sure python boto module is installed into your server (sudo pip install boto)
  • Configure the backend with Amazon IAM credentials.
  • Create a new record and save the details.

  • Enter the Authentication Key and Secret.
  • Enter the s3 bucket name.
  • Get a temporary source location where the zip will be saved temporary.

  • Go to Settings -> Automation -> Scheduled Actions
  • Configure the automated action in Scheduled action.

  • Enter the prefered time duration according to days, week, month when to trigger backups.
  • You can configure your backup every hour, minutes also.
  • An Easy solution to get everything back up from single click !!!
